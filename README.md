# seyam.codes - Personal Portfolio

Welcome to **seyam.codes**, my personal portfolio built with a focus on high-performance 3D graphics, smooth animations, and interactive storytelling.

## 🚀 Overview

This portfolio is not just a static page but an interactive experience. It showcases my skills, professional journey, and projects using modern web technologies.

## ✨ Features

- **Interactive 3D Environments:** Utilizes **React Three Fiber** and **Three.js** to create immersive 3D scenes.
- **Physics-Based Tech Stack:** A dynamic, interactive tech stack showcase powered by **React Three Rapier**.
- **Smooth GSAP Animations:** Professional-grade scroll animations and transitions using **GSAP** and **ScrollTrigger**.
- **Dynamic 3D Character:** A custom 3D character with real-time interactions and animations.
- **Modern UI/UX:** A clean, responsive design that works seamlessly across all devices.
- **Comprehensive Project Showcase:** Detailed sections for "What I Do", "Work", and "Career".

## 🛠️ Tech Stack

- **Framework:** [React](https://reactjs.org/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **3D Engine:** [Three.js](https://threejs.org/) / [@react-three/fiber](https://github.com/pmndrs/react-three-fiber)
- **Physics:** [@react-three/rapier](https://github.com/pmndrs/react-three-rapier)
- **Animations:** [GSAP](https://greensock.com/gsap/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Analytics:** [@vercel/analytics](https://vercel.com/analytics)

## 📦 Getting Started

### Prerequisites

- Node.js (Latest LTS version recommended)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/seyam-bhuiyan/First-Webpage.git
   ```

2. **Navigate to the project folder:**
   ```bash
   cd seyam.codes-main/seyam.codes-main
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

### Building for Production

To create a production-ready build:
```bash
npm run build
```

## 📂 Project Structure

- `src/components/`: React components for various sections (About, Work, Contact, etc.).
- `src/components/Character/`: Core 3D character logic and scenes.
- `src/components/styles/`: Component-specific CSS files.
- `public/models/`: 3D assets and environment maps.
- `public/images/`: Static images and icons.
